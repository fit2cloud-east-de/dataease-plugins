### 步骤一：登录 DataEase 系统

打开浏览器，访问 DataEase 系统并完成登录。

### 步骤二：选择 Oracle 数据源

在数据源管理页面，找到并点击 **Oracle** 图标，进入 Oracle 数据源配置页面。

### 步骤三：填写连接信息

在配置表单中填入以下信息：

| 参数          | 说明                           |
| ------------- | ------------------------------ |
| 主机名/IP地址 | 数据库服务器的主机名或 IP 地址 |
| 端口          | 数据库服务监听的端口号         |
| 数据库名称    | 目标数据库的名称               |
| 用户名        | 具有访问权限的数据库账号       |
| 密码          | 对应账号的登录密码             |
| 连接方式      | Oracle 数据库连接标识类型      |
| Schema        | 目标数据库 Schema 名称         |

填写完成后，点击**校验连接**按钮。校验通过即表示成功接入 Oracle 数据源。
