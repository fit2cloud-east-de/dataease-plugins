### 步骤一：登录 DataEase 系统

打开浏览器，访问 DataEase 系统并完成登录。

### 步骤二：选择 SAP HANA 数据源

在数据源管理页面，找到并点击 **SAP** 图标，进入 SAP HANA 数据源配置页面。

### 步骤三：填写连接信息

在配置表单中填入以下信息：

| 参数     | 说明                      |
|--------|---------------------------|
| IP 地址 | 数据库服务器的 IP 地址      |
| 端口号  | 数据库服务监听的端口号      |
| 用户名  | 具有访问权限的数据库账号    |
| 密码    | 对应账号的登录密码          |
| Schema | 目标数据库 Schema 名称     |

填写完成后，点击**校验连接**按钮。校验通过即表示成功接入 SAP HANA 数据源。

## 注意事项

- 请确保网络环境可以正常访问 SAP HANA 数据库服务器。
- 所填写的数据库账号需具备相应 Schema 的查询权限。
- 本插件仅支持 SAP HANA 2.0 及以上版本。
