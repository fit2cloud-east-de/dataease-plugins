### 步骤一：登录 DataEase 系统

打开浏览器，访问 DataEase 系统并完成登录。

### 步骤二：选择力导向图

在仪表板/数据大屏配置页面，找到并点击 **力导向图** 图标。

### 步骤三：准备数据

力导向图需要包含起点数据、终点数据以及边的标签数据：

| 字段         | 说明                       |
| ------------ | -------------------------- |
| SourceID     | 起始点的唯一标识           |
| SourceTeam   | 起始点所属的分组           |
| SourceLabel  | 起始点的标签               |
| SourceTip    | 起始点的提示信息（JSON格式）|
| TargetID     | 终点的唯一标识             |
| TargetTeam   | 终点所属的分组             |
| TargetLabel  | 终点的标签                 |
| TargetTip    | 终点的提示信息（JSON格式） |
| EdgeLabel    | 起点到终点的边标签         |

注：同一个节点的数据保持一致，根据 ID 唯一标识判断节点。

### 步骤四：配置样式

#### 标题样式
配置图表标题的样式属性。

#### 基础样式
- 节点配色方案：支持选择不同的配色方案，按分组顺序依次配色
- 不透明度、边框颜色、边线颜色、节点大小
- 自定义配置：边的宽度、中心引力、碰撞半径/强度、团队内/间连接距离与强度

注：当节点数据过多时，建议关闭自定义配置，并将节点大小调至10以内。

#### 背景样式
支持配置视图的内边距、圆角、背景模糊、背景颜色、背景图片、背景边框等。

#### 标签样式
- 节点标签：控制显示/隐藏，设置文本颜色、大小、位置（上/中/下/左/右）
- 边标签：控制显示/隐藏，设置文本颜色、大小

注：节点数据过多时，建议关闭节点标签和边标签。

#### 提示样式
- 开启提示时，鼠标点击节点/边触发提示窗口，展示 Tip 字段信息
- 支持关联的边和节点高亮
- 可设置提示窗文本颜色、大小、高度（30-1000）
