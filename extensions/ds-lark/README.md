### 步骤一：登录 DataEase 系统

打开浏览器，访问 DataEase 系统并完成登录。

### 步骤二：配置飞书应用对接

在使用飞书数据源前，需先完成飞书应用的平台对接配置。进入【系统设置】>【平台对接】，确保飞书应用已正确配置并启用。若该设置无效或被禁用，飞书数据源将无法使用。

### 步骤三：授权飞书多维表格

确认飞书多维表格已授权对应的飞书应用访问相关数据。根据权限需求，选择以下其中一种授权方式：

| 授权方式 | 操作人 | 权限说明 |
| -------- | ------ | -------- |
| 开通多维表格权限 | 企业管理员 | 仅提供只读权限 |
| 开通云文档权限 | 普通管理员 | 具备编辑权限 |

### 步骤四：获取 app_token

打开目标飞书多维表格，从浏览器地址栏 URL 中提取 `app_token`，其位于 URL 最后一个 `/` 与 `?` 之间。

### 步骤五：配置飞书数据源

在数据源配置页面中填入以下信息：

| 参数 | 说明 |
| ---- | ---- |
| app_token | 从多维表格 URL 中提取的访问令牌 |

填写 `app_token` 后，点击【获取数据表】和【获取视图】按钮，系统将自动加载该多维表格下的数据表和视图。

### 步骤六：设置定时同步

飞书数据源支持定时同步，操作方式与 API 数据源一致，可按需配置定时同步规则。

## 注意事项

- 飞书数据源依赖【平台对接】中的飞书应用配置，使用前请确保该配置有效且已启用。
- 使用前需确认飞书多维表格已对对应的飞书应用完成授权。
- 插件涉及的 API 接口包括：查询记录、列出视图、列出数据表。
